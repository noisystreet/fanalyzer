# 免责声明与数据使用说明

**最后更新：2026-05**

本文件适用于 **Fanalyzer**（crate 名 `fanalyzer`）项目及其发布的 CLI、Web 界面与相关文档（下称「本软件」）。使用本软件即表示你已阅读并理解以下内容。

---

## 1. 非投资建议

本软件输出的基金净值、排行、行业配置、重仓股、风险指标、筛选结果等，**仅供个人学习与研究参考**，**不构成**任何形式的投资建议、要约、承诺或保证。

- 本软件**不提供**证券、基金销售或投资顾问服务。
- 你应基于自身风险承受能力独立判断，必要时咨询持牌金融机构或专业人士。
- **基金有风险，投资需谨慎。** 历史业绩、回测或统计指标不代表未来表现。

---

## 2. 数据来源与归属

本软件通过公开网络访问第三方金融信息站点获取数据，主要包括（但不限于）：

| 类型 | 典型来源 |
|------|----------|
| 基金净值、基本信息 | 东方财富网 / 天天基金网（`fund.eastmoney.com`、`fundf10.eastmoney.com` 等） |
| 排行、行业配置、重仓股 | 同上及关联 API / 页面 |

- 数据**版权归原数据提供方所有**；本软件仅作技术层面的读取与展示。
- 本软件与东方财富、天天基金及其关联方**无隶属、合作或授权关系**，亦**不代表**其官方产品或观点。
- 本软件**不对**第三方数据的准确性、完整性、及时性作任何明示或默示担保。

---

## 3. 数据获取方式与合规使用

本软件通过 HTTP 客户端访问公开页面或接口，并在本地缓存部分结果以提升体验。

**你在使用本软件时，应当：**

1. **遵守中华人民共和国相关法律法规**（包括但不限于网络安全、数据安全、个人信息保护等规定）。
2. **遵守数据提供方的用户协议、服务条款与 robots 规则**；若第三方禁止自动化访问，你应停止相关用法或改用合规渠道。
3. **限于个人非商业研究用途**；不得将抓取结果用于商业转售、批量对外提供数据服务、或干扰目标站点正常运营。
4. **控制访问频率**：避免高频、并发或分布式抓取；本软件未内置大规模爬虫能力，请勿改造用于此类用途。
5. **不得绕过**目标站点的访问控制、验证码、付费墙或权限机制。

**禁止用途示例（非穷尽）：**

- 未经授权的数据再分发或 API 商业化
- 对第三方服务器施加明显超出正常用户行为的负载
- 将本软件用于任何违法或侵权活动

因违反上述规定或第三方条款而产生的责任，**由使用者自行承担**。

## 4. 数据与功能限制

- **接口与页面结构可能变更**，导致解析失败、字段缺失或数值错误；维护者无义务保证与官网实时一致。
- 季报类数据（行业配置、重仓股等）存在**披露滞后**；净值可能存在** T+1 更新**延迟。
- 分析指标（夏普、Alpha、Beta、回撤等）基于历史净值与简化模型计算，**口径可能与机构终端或基金合同表述不同**。
- 名称搜索、代码解析、离线缓存依赖历史访问记录，**可能过期或不完整**。

请在做出任何资金决策前，以**基金管理人公告、托管行披露及官方渠道**为准进行核实。

## 6. 隐私与本地数据

- 本软件在本地缓存基金代码映射与净值 JSON，路径见配置（默认用户缓存目录下的 `fanalyzer`）。
- 本软件**不会**主动向项目维护者上传你的查询记录；但若你自行将日志、缓存或截图公开分享，请注意其中可能包含个人使用痕迹。
- 请勿在配置文件、提交记录或 Issue 中粘贴 API Key、Token 等敏感信息（本项目当前亦**不应**需要此类密钥）。
